**eclipse**
https://blog.codecentric.de/en/2013/04/again-10-tips-on-java-debugging-with-eclipse/
ctrl + k, ctrl + shift +k for next/previous searched elements
ctrl+shift+R : search resource: files etcc
ctrl+shift+T : search type: class function etc
ctrl+shift+L : text search
ctrl+J: incremental search, typed is searched, press ctrl+J for next finding
ctrl+H: all search
ctrl+shift+: jump to matching brace
ctrl+shift+P: jump to matching any bracket
ctrl+shift+O: bring all imports
ctrl+1: quick fix window
ctrl+E: list of all open editors
ctrl+F6, ctrl+F7, ctrl+f8
ctrl+shift+F: beautify code
ctrl+T: shows implementation heirarchy
ctrl+3: search through actions/commands provided
place caret on variable, hover shows javadoc, shift+hover shows source code
ctrl+Q: jump to last edit location

refactoring:
alt+shift+R : select and change variable at mltiple locations

**sublime**
ctrl + shift + L: select block  of lines
Ctrl+D  for quick select next
Ctrl+U for select backwards
alt + f3 for all selections on current word

ctrl + M jump to closing brace

jsFormt for formating: alt + ctrl + f
ctrl + ` to open console
ctrl + shift + p => write install packages => search for packages to install

f12 : jump to definition
alt + - to jump back

ctrl +P file name search

**idea:**
ctrl + ` : for switching
Add/remove a selection: Alt + Shift + Mouse Click
ctrl + ] or [ to jump to opening closing braces
ctrl + shift + R : filename search
shift+shift search everywhere
ctrl + shift +a : do an action
jump to line: ctrl + L or navigate > gotoline

**clion:**
put caret on variable and press alt+f7 to find all usage 
use f3, shift+f3 to navigate through usage
ctrl+b to navigate to definition of class,funciton
parent-child relationship heirarchy: ctrl+H
quick definition: ctrl+shift+I


**intellij:**

shift+shift: find files
alt+ctrl+o : remove unused imports
f2 / shift+f2 : jump next/prev error
